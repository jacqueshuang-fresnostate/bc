# 客服会话使用 Semi Chat

## 背景

用户要求在线客服的会话使用 Semi UI 的 `Chat` 组件：`import { Chat } from '@douyinfe/semi-ui';`。当前“在线客服”页面的消息记录使用手写卡片列表展示，不够贴近会话组件语义。

## 目标

- 在线客服会话详情中的消息流改用 Semi UI `Chat` 组件渲染。
- 保留现有会话列表、创建会话、状态维护、客服分配和后台回复表单。
- 不修改后端接口、数据模型和消息保存逻辑。
- 保持用户消息、管理员消息、系统消息的展示区分。

## 非目标

- 不实现实时聊天、WebSocket、已读回执或用户端客服入口。
- 不改动客服消息发送 API 的请求/响应格式。
- 不重构整页布局。

## 验收标准

- `SupportManagementPage` 从 `@douyinfe/semi-ui` 导入并使用 `Chat`。
- 选中客服会话后，消息记录由 `Chat` 组件展示。
- 后台回复输入和提交逻辑保持可用。
- `npm run build` 通过。
- `架构设计.md` 和 `TODO.md` 记录本次调整。
