# 后台彩种控制台

## 背景

用户要求后台需要有一个“彩种控制台”，实时显示每一个彩种的倒计时和开奖号码。当前后台已有彩种管理、开奖期号与开奖源页面，但缺少一个运营总览式的实时控制台来按彩种扫描封盘/开奖倒计时和最新开奖结果。

## 目标

- 新增“彩种控制台”后台页面。
- 页面按每个彩种展示实时状态：
  - 彩种名称、号码类型、开奖模式、销售状态。
  - 当前 open 期号或最近相关期号。
  - 距离封盘倒计时。
  - 距离开奖倒计时。
  - 当前/最近开奖号码。
  - 最近已开奖期号及开奖时间。
- 倒计时每秒在前端刷新。
- 数据定时刷新，确保新增期号或开奖结果能回流到控制台。
- 后台导航和 dashboard 模块入口能进入该页面。
- 更新 `架构设计.md` 和 `TODO.md`。

## 非目标

- 不新增 WebSocket 或服务端推送。
- 不改动后端开奖期号接口。
- 不修改自动开奖/调度逻辑。
- 不实现手机端控制台。

## 验收标准

- 后台侧边栏和 dashboard 能看到“彩种控制台”入口。
- 页面展示所有彩种，即使某个彩种暂无 open 期号。
- open 期号显示封盘倒计时和开奖倒计时，倒计时每秒变化。
- 已开奖彩种显示最近开奖号码，开奖号码保持英文逗号分隔格式。
- `npm run build` 通过。
- 浏览器验证页面能打开且无功能性控制台错误。
