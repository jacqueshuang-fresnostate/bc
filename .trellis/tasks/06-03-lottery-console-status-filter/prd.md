# 彩种控制台状态筛选

## 背景

彩种控制台当前按彩种展示销售状态、当前 open/closed 期号、倒计时和最近开奖号码，但所有彩种卡片一次性展示。运营想快速查看某一种状态，例如销售中、已停售、开盘中、待开奖、已开奖或暂无当前期。

## 目标

1. 彩种控制台增加状态筛选入口。
2. 筛选在前端本地完成，不新增后端接口。
3. 支持以下筛选：
   - 全部
   - 销售开启
   - 已停售
   - 开盘中
   - 待开奖
   - 已开奖
   - 无当前期
4. 筛选项显示对应数量，当前筛选高亮。
5. 筛选后卡片列表只展示匹配彩种；无匹配项时展示空状态。
6. 保持后台控制台紧凑、可扫描的视觉风格。

## 范围

- 修改 `admin/src/pages/LotteryConsolePage.tsx`。
- 更新 `架构设计.md` 和 `TODO.md`。

## 验收标准

- `npm run build` 通过。
- 点击不同筛选项后，页面展示数量和卡片列表同步变化。
- 默认进入页面展示全部彩种。
